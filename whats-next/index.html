What's Next?

We have docs! But what now? Here are some resources to help you continue further on your documentation journey!

The Good Docs Project

Check out The Good Docs Project's resources! Their material was drawn on heavily in this short guide, but there is so much more available, from more templates to an example of great documentation for a fictional tool.

Just the Docs

Additionally, Just the Docs, the documentation site template we leveraged in this guide, has a useful documentation site where you can learn more about what the template can do, as well as how it can be configured and customized.

Documentation Writing

There's also loads of resources out there on how to improve your documentation writing skills. Write the Docs has a comprehensive documentation guide that's worth checking out, as it contains great information about what it means to write good documentation, and also contains links to a number of other resources worth learning about, such as style and accessibility guides.

Farewell, Friend!

Hopefully, this guide has been able to help you get started on having documentation for your own projects! Even if it may not be much right now, putting the building blocks in place will make it easier to add in necessary documentation as your project expands and grows. Best of luck!
